# [NotebookLM 검토용] 주식 퀀트 매매 자동화 시스템 프로젝트 현황 및 토의 안건

본 문서는 현재 개발 중인 "HTS 연동형 퀀트(Quant) 매매 보조 시스템"의 아키텍처와 현재 직면한 기술적 이슈를 정리한 문서입니다. 이 문서를 바탕으로 AI와 토의하여 해결책과 데이터 설계 방향을 도출하고자 합니다.

---

## 1. 프로젝트 개요 (Project Overview)
* **목적**: 키움증권 HTS에서 종목을 클릭할 때마다, 해당 종목의 기술적 지표(장기 이평선 등)와 재무/공시 데이터, 계좌 자금 현황을 0.1초 만에 백그라운드에서 분석하여 텍스트로 보고해 주는 '초고속 퀀트 보조 시스템' 구축.
* **주요 매매 기법**: 농사매매 (현금 비중 30% 유지, 112/224일선 기반 밥그릇 패턴 포착, -20% 하락 시 1-1-2-4 마틴게일 분할매수, 3년 연속 적자 및 부채 150% 이상 기업 필터링)
* **개발 원칙**: 화려한 UI 배제, 오직 정확한 Raw Data(JSON/텍스트) 출력에 집중.

## 2. 시스템 아키텍처 및 현재 구현 상태 (Current Status)
* **1. HTS 종목 인식 엔진 (완료)**: `win32gui`를 사용하여 키움증권 영웅문 HTS 창의 제목 변경을 감지하고, 정규식을 통해 순수 '종목명'만 추출하는 로직 구현 완료.
* **2. 종목명 ↔ 코드 매핑 시스템 (완료)**: KRX 상장사(2,873개) 종목코드 및 DART 고유번호(3,945개) JSON 매핑 사전 구축 완료.
* **3. DART 공시 API 연동 (완료)**: 종목 감지 시 DART API를 비동기로 호출하여 최근 90일 내 주요 공시 5건을 정상 수신 중.
* **4. 퀀트 차트 지표 연산 엔진 (완료)**: `FinanceDataReader`로 과거 900일 치 일봉(OHLCV) 데이터를 수집하고, `pandas-ta`를 통해 112일/224일/448일 이동평균선, 볼린저밴드 하단, 일목균형표 구름대를 완벽하게 수학적으로 연산 완료.
* **5. 웹소켓 프론트엔드 (완료)**: HTS 감지 -> 파이썬 비동기 연산 -> 브라우저(`index.html`)로 Raw JSON 텍스트 전송 파이프라인 완벽 가동 중.

## 3. 핵심 토의 안건 및 해결 요망 사항 (Discussion Points)
NotebookLM은 아래의 기술적 이슈와 기획 안건에 대해 분석하고 최적의 답변을 제안해 주시기 바랍니다.

### Issue A: 키움증권 REST API '현재가 조회' 500 에러 원인 및 TR 양식 분석
* **상황**: 현재 키움증권 실전투자 REST API(`https://api.kiwoom.com`)에 접근하여 OAuth2 Access Token을 발급받는 데까지는 성공했습니다.
* **문제**: 발급받은 토큰으로 '국내주식 현재가 조회' 엔드포인트를 찔렀으나 `500 INTERNAL_SERVER_ERROR (예기치 못한 에러가 발생하였습니다)`가 반환됨.
* **원인 추정**: 한국투자증권의 TR ID인 `FHKST01010100`을 그대로 차용하여 파라미터(`fid_cond_mrkt_div_code`, `fid_input_iscd`)를 넘겼으나, 키움증권 서버에서 요구하는 특정 TR ID 양식이나 필수 헤더(예: Mac Address 등)가 누락되었을 확률이 높음.
* **NotebookLM에게 요청하는 답변**: 키움증권 OpenAPI/REST API 공식 문서 가이드를 바탕으로, **국내주식 단일 종목의 현재가(시세)를 REST API로 조회할 때 정확히 어떤 TR ID와 파라미터 구조(JSON/Headers)를 사용해야 하는지** 찾아주세요.

### Issue B: 추출할 핵심 데이터 필드(Data Schema) 최적화 설계
* **상황**: 방대한 시세 및 재무 데이터 중, 트레이더가 '농사매매'에 직관적으로 활용할 수 있는 핵심 알짜배기 데이터만 JSON으로 추출하려 함.
* **NotebookLM에게 요청하는 답변**: 성공적인 농사매매 및 마틴게일 연산을 위해 반드시 화면에 띄워야 하는 **데이터 항목(필드) 리스트를 정의해 주세요.** 
* *(예시: 현재가, 전일대비 등락률, 224일선 이격도, DART 부채비율, 통계좌 예수금 비중, 마틴게일 2차 진입 요구 금액 등)*
