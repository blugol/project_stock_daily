# ✅ 퀀트 자동화 시스템: 목표 데이터 추출 체크리스트 (Task)

고객님께서 노트북LM과 토의하여 확정하신 **'농사매매 필수 데이터 15종 + 계좌 매매 기록'**입니다. 현재는 연동(실행)하지 않고 목표 스펙(Task)으로만 영구 기록해 두었으며, 추후 고객님의 개발 지시가 떨어지면 이 체크리스트를 바탕으로 하나씩 API를 연동해 나갈 것입니다.

---

## 1. 👥 주주현황 및 지분 구성 (세력 빈집 판별용)
**[수집 소스: DART API (대량보유상황보고서 및 사업보고서)]**
- `[ ]` **최대주주 및 특수관계인 지분율 (%)**: 기업의 실제 유통물량과 '세력 빈집' 상태 파악
- `[ ]` **자사주 및 자사주신탁 지분율 (%)**: 시장 유통물량 축소 및 기업의 주가 방어 의지 확인
- `[ ]` **5% 이상 주요 주주 지분율 (%)**: 투자조합, 외국인 등 스마트 머니 개입 여부 및 전체 리스트 파악

## 2. 📉 실전 재무 필터링 (깡통 기업 컷오프용)
**[수집 소스: DART API (단일회사 주요계정 및 재무제표)]**
- `[ ]` **영업이익 및 당기순이익**: 기업 흑자 여부 및 '4년 연속 적자' 상폐 리스크 필터링
- `[ ]` **매출액 추이**: 연 매출 30억 미만 상폐 요건 해당 여부 및 성장 추세 확인
- `[ ]` **부채비율 및 자본잠식률**: 부채비율 150% 이하 요건 검증 및 완전/부분 자본잠식 원천 차단
- `[ ]` **유보율 (%)**: 자본금 대비 현금 보유량 파악 (무상증자, 배당 여력 확인)
- `[ ]` **이자보상배율**: 1 미만 지속 시 은행 이자도 못 내는 '좀비 기업'으로 판별하여 제외
- `[ ]` **대출채권(매출채권) 및 재고자산 비중**: 악성 재고 및 떼일 돈(외상값)이 과도한 숨은 리스크 필터링

## 3. ⚖️ 기업 가치평가 보조 지표 (안전 마진 확보용)
**[수집 소스: 키움 OpenAPI (기업 펀더멘털 조회)]**
- `[ ]` **PBR (주당순자산비율)**: 장부상 가치 대비 주가가 싼지 (1 이하) 확인
- `[ ]` **PER (주가수익비율)**: 주가 대비 이익 창출력이 좋은지 (10 이하) 확인
- `[ ]` **EV/EBITDA**: 기업가치 대비 영업현금흐름 (5 이하) 확인
- `[ ]` **ROE / ROA**: 자기자본 및 총자산 대비 수익률 (5 이상) 확인

## 4. 💼 내 계좌 및 매매 기록 (실전 마틴게일 자금 관리용)
**[수집 소스: 키움 OpenAPI (계좌 잔고 및 거래내역 조회)]**
- `[ ]` **매매 단가 (매수 평균가)**: 해당 종목을 얼마에 샀는지 평단가 파악
- `[ ]` **보유 수량 (매입 주식 수)**: 현재 해당 종목을 몇 주 들고 있는지 파악
- `[ ]` **매입 금액 (총 매수금)**: 해당 종목에 내 시드가 총 얼마 들어갔는지 파악 (마틴게일 1-1-2-4 비중 계산용)
- `[ ]` **매매 일자 (최종 거래일)**: 해당 종목을 마지막으로 언제 샀는지/팔았는지 날짜 파악

---

> **개발 유의사항 (원칙 준수)**
> 위 리스트는 오직 **'텍스트(Raw JSON) 출력'** 목적으로만 수집되며, 어떠한 디자인 요소도 추가하지 않습니다. 고객님의 허락(Run 명령)이 떨어지기 전까지는 API 호출 로직을 절대 코드에 반영하지 않습니다.
