# API 키 점검 및 종합 활용 분석 보고서

본 보고서는 고객님께서 제공해주신 `.env` 파일 내부의 **4가지 키**가 각각 어떤 역할을 하며, 기술적으로 어떻게 쓰이는지 단 하나도 빠짐없이 완벽하게 분석한 문서입니다.

---

## 🔑 1. DART_API_KEY (금융감독원 전자공시 마스터 키)
* **상태**: 정상 작동 확인 완료 (삼성전자 데이터 호출 성공)
* **역할**: 국가 기관인 금융감독원(DART) 서버의 방화벽을 통과하게 해주는 핵심 출입증입니다.
* **이 키로 할 수 있는 일 (수신 데이터)**:
  * **공시/뉴스**: 특정 종목의 최근 공시 리스트, 주주총회 소집 결의, 유상증자 정보 등
  * **재무 정보**: 매출액, 영업이익, 당기순이익, 자산총계 등 분기별/연도별 핵심 재무 지표
  * **투자 지표**: 임원 지분 변동, 고배당 기업의 주당 배당금 및 배당 수익률 등
* **파이썬 내부 작동 방식**: HTS에서 종목이 감지되면, 파이썬 서버가 이 키를 가지고 DART 서버에 HTTP 요청(GET)을 날려 위 정보들을 JSON(텍스트 데이터) 형태로 긁어옵니다.

---

## 🔑 2. KIWOOM_APP_KEY (증권사 접속 아이디)
* **상태**: 키 형태 및 포맷 정상 확인 완료
* **역할**: 증권사 OpenAPI(REST API) 시스템에 접속할 때 필요한 **프로그램(앱) 전용 아이디(ID)**입니다. 증권사가 "어떤 프로그램이 시세를 요청하는지" 식별하는 데 쓰입니다.
* **이 키로 할 수 있는 일**:
  * 단독으로는 쓸 수 없으며, 3번(SECRET_KEY)과 짝을 이뤄 **접근 토큰(Access Token)**을 발급받는 데 사용됩니다.

---

## 🔑 3. KIWOOM_SECRET_KEY (증권사 접속 암호)
* **상태**: 키 형태 및 포맷 정상 확인 완료
* **역할**: 증권사 OpenAPI에 접속할 때 사용하는 **프로그램(앱) 전용 특수 비밀번호**입니다. 해커가 고객님의 프로그램인 척 위장하는 것을 막아주는 가장 강력한 보안 자물쇠입니다.
* **파이썬 내부 작동 방식 (2번 + 3번 키의 조합)**:
  * 파이썬 서버가 켜질 때, `APP_KEY`와 `SECRET_KEY`를 묶어서 증권사 인증 서버(`.../oauth2/token`)에 보냅니다(POST 요청).
  * 증권사는 두 키가 일치하면 **24시간 동안 유효한 임시 출입증(Access Token)**을 파이썬 서버에 발급해 줍니다.
  * 파이썬은 이 임시 출입증을 이용해 **현재가, 10호가 잔량, 거래량, PER, PBR** 등 엄청난 양의 실시간 시세 데이터를 웹소켓으로 긁어올 수 있게 됩니다.

---

## 🔑 4. KIWOOM_ACCOUNT_NO (실제 계좌번호)
* **상태**: 정상 확인 완료 (10자리 숫자 포맷)
* **역할**: 실제 돈이 오가거나, 고객님의 자산 내역을 열람할 때 타겟이 되는 종착점입니다.
* **이 키로 할 수 있는 일 (수신 및 실행 데이터)**:
  * **잔고 조회**: 현재 계좌에 있는 현금(예수금), 보유 주식 종목 리스트, 총 매입 금액, 총 평가 금액, **현재 수익률(%)** 등을 조회.
  * **자동 매매(주문)**: 시세를 보다가 타점이 왔을 때 파이썬이 이 계좌번호를 통해 "삼성전자 10주 시장가 매수!" 같은 **실제 주문(Order)**을 증권사 서버로 쏠 수 있게 해줍니다.

---

## 🎯 종합 결론 (4개 키의 시너지)
고객님 컴퓨터의 파이썬 서버는 이 4개의 키를 다음과 같이 조합해서 사용합니다.

1. HTS에서 **코스모화학**을 누름.
2. 파이썬 서버가 `DART_API_KEY` (1번 키)를 써서 코스모화학의 **최근 공시와 영업이익**을 긁어옴.
3. 파이썬 서버가 `APP_KEY`와 `SECRET_KEY` (2번, 3번 키)로 뚫어둔 통로를 통해 코스모화학의 **실시간 호가와 현재가**를 긁어옴.
4. 파이썬 서버가 `ACCOUNT_NO` (4번 키)를 통해 코스모화학을 우리가 몇 주나 갖고 있고 수익률이 몇 프로인지 긁어옴.
5. 이 모든 데이터가 0.1초 만에 합쳐져서 웹페이지 대시보드에 화려하게 렌더링 됨.

이러한 완벽한 데이터 연동이 가능하도록 준비가 완료된 상태입니다.
