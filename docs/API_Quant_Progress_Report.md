# API 및 퀀트 지표 연동 경과 보고서 (API_Quant_Progress_Report)

본 보고서는 금일 테스트 및 연동에 성공한 API 활용 내역과 퀀트 엔진의 가동 결과를 빠짐없이 기록하여, 내일 또는 추후 작업 시 즉시 이어서 개발할 수 있도록 작성되었습니다.

---

## 1. DART 공시 API 연동 완료 (100%)
* **역할**: 기업의 최신 공시 내역(유상증자, 분기보고서, 단일판매공급계약 등)을 가져옵니다.
* **사용된 키**: DART API Key (`.env` 파일 내 `DART_API_KEY`)
* **구현 내역**:
  * 한국거래소(KRX) 상장 종목 3,000여 개의 종목명을 DART 전용 고유번호(8자리)로 변환하는 매핑 데이터 생성 완료 (`app/dart_codes.json`)
  * `DartClient` 모듈을 통해 특정 기업의 최근 90일 내 주요 공시 5개를 가져오는 비동기 호출 테스트 완벽 성공. (예: 엘오티베큠의 분기보고서 정상 수신 확인)

## 2. 퀀트 지표(단테 농사매매 기법) 연산 엔진 구축 완료 (100%)
* **역할**: 시세만 보는 것이 아니라, 과거 2년 치 차트를 분석해 세력주/급등주 패턴을 찾아냅니다.
* **사용 기술**: `FinanceDataReader`(데이터 수집), `pandas`, `pandas-ta`(수학 연산)
* **구현 내역**:
  * HTS에서 종목이 감지되면, 해당 종목의 과거 900일 치 일봉(OHLCV)을 백그라운드에서 즉시 긁어옵니다.
  * 단테님 기법의 핵심인 장기 이평선 계산 완벽 성공: **112일선 (반년), 224일선 (1년), 448일선 (2년)**
  * 추세 및 저항 확인을 위한 **볼린저밴드 하단선(20,2)** 및 **일목균형표(구름대 선행스팬 A/B)** 소수점 2자리까지 오차 없이 연산 완료 확인.

## 3. 키움증권 REST API 연동 현황 (진행 중 - 50% 완료)
* **역할**: 실시간 현재가, 체결량, PER/PBR 등의 펀더멘털 데이터 및 내 계좌 정보를 가져옵니다.
* **사용된 키**: Kiwoom App Key, Secret Key, Account No
* **구현 내역**:
  * 키움증권 실전투자 서버(`https://api.kiwoom.com`)에 접근하여 24시간 유효한 엑세스 토큰(Access Token)을 발급받는 자동 인증 로직 구축 및 테스트 대성공.
* **해결해야 할 이슈 (내일 이어서 진행할 사항)**:
  * 토큰 발급 후, 현재가를 조회할 때 사용하는 파라미터 `tr_id` 양식이 키움 서버와 미세하게 맞지 않아 `500 INTERNAL_SERVER_ERROR`가 발생함.
  * **내일 조치 사항**: 키움 OpenAPI 문서에서 정확한 `tr_id` 명칭과 헤더 규격(예: Mac Address 필수 여부 등)을 확인하여 단어 하나만 수정해주면 즉시 해결될 예정.
